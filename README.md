# Coming Soon — Portfolio (scaffold)

This is a small, accessible "Coming Soon" landing page scaffold. It includes proper meta tags, alt text for images, header tags, and a simple email capture demo.

Files created:

- `index.html` — main page with meta tags and accessible markup
- `styles.css` — responsive styles and utility classes
- `script.js` — countdown and local email capture
- `images/placeholder.svg` — accessible placeholder logo

Quick start (PowerShell):

```powershell
# From the project folder, run a simple static server (Python must be installed):
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```

Notes:
- Replace `Your Name`, the `og:`/`twitter:` meta values and `images/placeholder.svg` with your real content before publishing.
- For production use, hook the form to an email service (Mailchimp, ConvertKit, etc.) or a backend endpoint.
